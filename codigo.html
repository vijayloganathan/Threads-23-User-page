CODIGO

Codigo is an event where participants compete to solve coding challenges. It is a way to see how much someone knows about computer programming.

RULES

  • No external devices are allowed to use, if caught you are disqualified
  • Any programming language can be used.
  • Solo or Duo in a team

ROUNDS

  • 1st round: 30 MCQ based technical questions (30mins)
