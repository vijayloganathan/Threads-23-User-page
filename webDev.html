WEB DEVELOPMENT

Are you new to web development and eager to learn how to build your very own web app? Our one-day web development workshop is the perfect opportunity for you to learn the fundamentals of web development. Create your own simple one-page web apps using vanilla JavaScript. And that's not all - you'll also learn how to create a URL shortener app using Python Flask!

24/02/2023, 10AM - 4PM

Updating Soon

DESCRIPTION

You'll get hands-on experience with the essential tools and techniques used by professional web developers, as we guide you every step of the way through the development process.

You'll learn how to design responsive user interfaces, code dynamic functionality using vanilla JavaScript, and build a URL shortener app using Python Flask. By the end of the workshop, you'll have a working web app and a powerful tool that you can use for your own projects.

Whether you're looking to start a career in web development or simply want to gain the skills needed to create your own web app, this workshop is perfect for you! Our experienced instructors will provide you with the guidance and knowledge you need to succeed. Don't miss this opportunity to learn the essentials of web development and build your own web app. Enroll now and unlock the potential of your creativity!
